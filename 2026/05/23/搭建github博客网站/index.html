搭建github博客网站

###一、整体原理（先建立概念）

GitHub Pages 本质上是：GitHub 免费帮你托管静态网页。
也就是说：你写Markdown文章，然后Hexo/Hugo/Jekyll
帮你生成HTML网页再推送到 GitHub。GitHub 自动帮你变成博客网站。最终你会得到：
https://你的用户名.github.io
比如：
https://abc.github.io

以后写博客流程（非常简单）

以后你只需要：

写文章
hexo new “kv-cache”
本地预览
hexo server
发布
hexo clean
hexo generate
hexo deploy
